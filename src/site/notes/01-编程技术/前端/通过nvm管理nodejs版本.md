# 常用命令

``` shell
# 查看所有可用版本
nvm ls available

# 安装指定版本
nvm install 18.17.1

# 查看已安装版本
nvm list

# 应用版本
nvm use 18.17.1

# 查看node版本
node -v

# 删除指定版本
nvm uninstall 18.17.1
```
